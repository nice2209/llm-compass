# 용도별 LLM 추천

> 🚧 **준비 중** — 이 기능은 데이터 축적 후(서비스 시작 14일 이후) 활성화됩니다.

예산과 목적에 맞는 최적의 LLM 조합을 추천하는 기능입니다. 다음과 같은 용도별 시나리오를 다룰 예정입니다.

- **코딩 어시스턴트** — 정확도 우선, 컨텍스트 윈도우 큰 모델
- **대량 요약/분류** — 입력 단가 최저 모델
- **챗봇/대화** — 입출력 균형 + 빠른 응답
- **에이전트/툴 호출** — 함수 호출 안정성 + 비용 효율
- **저예산 실험** — 무료/초저가 티어

그동안은 [가격 비교표](/pricing/)에서 직접 정렬·필터하여 원하는 모델을 찾을 수 있습니다.
