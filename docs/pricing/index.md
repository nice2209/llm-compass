# LLM API 가격 비교

주요 LLM 제공업체의 API 가격을 한눈에 비교합니다. 가격은 **USD / 1M 토큰** 기준으로 정규화되어 있으며, 단위 토글로 1K 토큰 기준으로도 볼 수 있습니다.

- **검색**: 모델명 또는 제공업체로 필터
- **정렬**: 입력 단가 / 출력 단가 / 컨텍스트 윈도우 헤더 클릭
- **필터**: 제공업체 뱃지를 클릭해 선택 (선택 없으면 전체)
- **CSV 내보내기**: 현재 표시 중인 행을 CSV로 다운로드
- **최저가 하이라이트**: 입력 단가 최저 모델 강조

::: tip 데이터 갱신
가격 데이터는 GitHub Actions가 매일 공식 페이지에서 수집해 자동 갱신합니다. 일부 제공업체는 페이지 구조 변동·인증 요구로 일시적으로 비어 있을 수 있습니다.
:::
